# Dotfiles Plus Homebrew Tap

This branch contains the Homebrew formula for Dotfiles Plus.

## Installation

```bash
brew tap anivar/dotfiles-plus https://github.com/anivar/dotfiles-plus
brew install dotfiles-plus
```

## What is Dotfiles Plus?

Dotfiles Plus is an AI-powered shell enhancement framework that brings the power of AI directly into your terminal. It works with Bash 5+, Zsh, and integrates with Claude, ChatGPT, Gemini, and other AI providers.

## Formula Details

- **Version**: 2.0.1
- **Dependencies**: Bash 5.0+, jq (recommended)
- **License**: MIT
